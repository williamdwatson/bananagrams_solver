Generates random letters based on user input

§Arguments

  • what - Whether to generate characters from an “infinite set” (i.e. all are equal likelihood), or selected from “standard Bananagrams” (144 tiles) or “double Bananagrams” (288 tiles)
  • how_many - How many tiles to randomly generate; must be greater than 0, and less than 144 for regular Bananagrams, or 288 for double

§Returns

Result of mapping of each uppercase Latin character to the number of times it’s present

or String Err upon failure
